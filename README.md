## Project Overview

This project contains:
1. **`movies_eda.ipynb`** — Jupyter Notebook with full Exploratory Data Analysis (EDA) on the MovieLens `movies.csv` dataset.
2. **`app.py`** — Streamlit web application to explore movies by genre.
3. **`movies_cleaned.csv`** — Cleaned dataset output from the EDA (generated by the notebook).

---

## Setup & Installation

```bash
pip install pandas numpy matplotlib seaborn streamlit
```

---

## Running the Notebook

1. Place `movies.csv` in the same folder as the notebook.
2. Open the notebook:
   ```bash
   jupyter notebook movies_eda.ipynb
   ```
3. Run all cells. The cleaned CSV (`movies_cleaned.csv`) will be generated automatically.

---

## Running the Streamlit App

1. Make sure `movies_cleaned.csv` is in the same directory as `app.py`.
2. Start the app:
   ```bash
   streamlit run app.py
   ```
3. Open your browser at `http://localhost:8501`

---

## Deploying to Streamlit Community Cloud

1. Push this project to a **GitHub repository**.
2. Go to [share.streamlit.io](https://share.streamlit.io) and connect your repo.
3. Set the main file path to `app.py`.
4. Make sure `movies_cleaned.csv` is committed to the repo, or add a `requirements.txt`.

### `requirements.txt`
```
pandas
streamlit
```

---

## App Features

- 🎭 **Genre selectbox** — choose one genre at a time
- 📋 **Results table** — all movies matching that genre, sorted by year (newest first)
- 🔍 **Title search** — filter results within a genre
- 📊 **Genre overview chart** — shown on the landing screen
- 📌 Sidebar with dataset stats
